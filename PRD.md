# PRD - Markdown Editor

## 文档信息

- 当前版本：v2.5.1
- 最近更新：2026-02-17

## 功能描述

### 已有核心功能
- Markdown 编辑、实时预览与 PDF 下载（分页/单页）。
- 历史记录保存、恢复、删除与清空。
- 预览内容复制与应用内提示反馈。

### 本次更新（v2.5.1）

- 预览页识别 `#ffffff` 形式的十六进制颜色代码，并在左侧渲染 10x10 SVG 色块。
- 颜色码与色块垂直居中并保持 4px 固定间距，保证阅读时视觉对齐。
- 点击色块可复制对应颜色值，悬浮时通过 tooltip 显示“点击复制颜色值”。


## 用户故事
- 作为用户，我希望在预览中看到颜色值旁的可视色块，以便快速确认颜色。
- 作为用户，我希望点击色块即可复制颜色值，并能通过悬浮提示明确交互意图。

## 技术方案
- 在预览模板注入颜色增强逻辑，遍历文本节点匹配 `#[0-9a-fA-F]{6}` 并将命中片段替换为“色块按钮 + 颜色文本”的内联结构。
- 色块使用 SVG 绘制，尺寸固定为 `10x10`，并通过样式确保与颜色文本垂直居中及 `4px` 间距。
- 点击色块时优先使用 Clipboard API 复制颜色值，回退到 `execCommand('copy')`；复制失败时复用现有错误 Toast。
- 按语义化版本规则升级版本号到 `2.5.1`，同步更新变更日志与版本常量。


## 测试用例
1. 在编辑器输入 `#ffffff` 并打开预览页，确认颜色值左侧显示 10x10 色块。
2. 确认色块与颜色文本垂直居中，且二者间距为 4px。
3. 悬浮色块显示 tooltip，点击后可在剪贴板中粘贴出对应颜色值。
4. 运行构建命令，确认本次改动未引入 TypeScript 编译错误。
