# PRD - Markdown Editor

## 文档信息
- 当前版本：v2.4.3
- 最近更新：2026-02-13

## 功能描述

### 已有核心功能
- Markdown 编辑、实时预览与 PDF 下载（分页/单页）。
- 历史记录保存、恢复、删除与清空。
- 预览内容复制与应用内提示反馈。

### 本次更新（v2.4.3）
- 历史记录界面文字字体统一为与编辑页“预览/下载”按钮相同字体（Times New Roman）。
- 预览界面文字字体统一为与编辑页“预览/下载”按钮相同字体，包含工具按钮与正文区域。

## 用户故事
- 作为用户，我希望历史记录面板与预览页面的文字风格一致，以获得统一的视觉体验。
- 作为用户，我希望预览页与编辑页关键按钮的字体统一，减少界面风格割裂感。

## 设计稿链接
- 本次为样式一致性微调，暂无独立设计稿。
- 建议后续补充 Figma 页面并在此处登记链接。

## 技术方案
- 在主样式文件中为历史记录面板设置字体族，并让历史相关按钮继承该字体。
- 在预览模板样式中为 `body` 与预览页按钮设置统一字体。
- 在配置常量中将预览字体配置调整为与按钮字体一致，保证单点配置可复用。
- 按语义化版本规则升级版本号到 `2.4.3`，同步更新变更日志。

## 测试用例
1. 打开历史记录面板，检查标题、分组标题、条目内容与按钮字体均为 Times New Roman 风格。
2. 点击“预览”打开新页，检查页面正文、工具按钮字体与编辑页“预览/下载”按钮一致。
3. 回归验证历史记录功能（新增、恢复、删除、清空）行为不受本次样式改动影响。
4. 运行自动化测试，确认现有历史记录测试全部通过。
