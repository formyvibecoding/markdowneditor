# PRD - Markdown Editor

## 文档信息
- 当前版本：v2.4.4
- 最近更新：2026-02-17

## 功能描述

### 已有核心功能
- Markdown 编辑、实时预览与 PDF 下载（分页/单页）。
- 历史记录保存、恢复、删除与清空。
- 预览内容复制与应用内提示反馈。

### 本次更新（v2.4.4）
- 预览页 Markdown 表格在右上角新增“复制表格”按钮。
- 复制按钮默认隐藏，鼠标悬浮到表格区域时显示，减少页面静态噪音。
- 点击后复制当前表格内容（HTML + 纯文本），复制成功后按钮短暂显示“已复制”。


## 用户故事
- 作为用户，我希望在预览表格附近可以快速复制当前表格，避免手动框选内容。
- 作为用户，我希望复制按钮只在悬浮时出现，从而保持阅读时界面简洁。


## 设计稿链接
- 本次为样式一致性微调，暂无独立设计稿。

## 技术方案
- 在预览模板中为每个渲染后的 Markdown `table` 自动包裹容器，并在容器右上角注入“复制表格”按钮。
- 通过 CSS 控制按钮仅在容器 hover/focus 时显示，确保符合“悬浮可见”的交互要求。
- 点击按钮时优先使用 `ClipboardItem` 复制 HTML + 纯文本，回退到纯文本复制；失败时复用现有错误 Toast。
- 按语义化版本规则升级版本号到 `2.4.4`，同步更新变更日志。


## 测试用例
1. 在编辑器输入包含 Markdown 表格的内容并打开预览页，确认每个表格右上角存在复制按钮。
2. 默认不悬浮时按钮不可见，鼠标移动到表格区域后按钮显示。
3. 点击“复制表格”后按钮短暂显示“已复制”，并可将表格内容粘贴到支持富文本的目标应用。
4. 运行构建命令，确认本次改动未引入 TypeScript 编译错误。
